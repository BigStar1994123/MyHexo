## Remove mongodb field from collections ##


```
{ 
    name: 'book',
    tags: {
        words: ['abc','123'],
        lat: 33,
        long: 22
    }
}
```

db.example.update({}, {$unset: {tags.words:1}}, false, true);  
remove words

```
  { 
      name: 'book',
      tags: {
          lat: 33,
          long: 22

  }
```
